# Protocol

A granular productivity tool — a fully local desktop app that ties together a
journal, hierarchical goals, and a drag-and-drop calendar so every goal turns
into concrete, scheduled tasks.

I built this for myself to plan my own work.

## Install

**[Download the latest installer (Windows)](https://github.com/dylaneulgen/protocol/releases/latest)**

Your data is local.

## Run from source

```bash
npm install
npm start
```
